# finplot/examples 目录下所有脚本功能总结

finplot 是一款高性能的Python金融数据可视化库，专注于金融图表绘制、回测辅助，支持股票、加密货币等多种金融数据的多维度可视化，具备流畅的交互体验和丰富的图表样式。其 examples 目录下的脚本均为官方提供的功能演示案例，覆盖基础绘图、实时数据、高级可视化、交互动效等核心场景，以下按脚本名称逐一总结其具体功能作用：

## 一、核心分析类脚本（基础数据可视化与指标分析）

- **analyze.py**：核心功能是金融数据多维度分析与可视化演示，支持加载价格数据（如股票、加密货币），绘制K线图、均线（EMA）、布林带、RSI指标、成交量、资金流向（Accum/Dist）、能量潮（OBV）等常用技术分析图表，同时整合多子图布局，可直观呈现价格走势与各类技术指标的联动关系，是finplot基础分析功能的核心演示脚本，也是量化分析入门常用案例。

- **analyze-2.py**：analyze.py 的补充优化版本，聚焦长期行情分析，支持对数坐标显示价格（适配长期大幅波动的行情），绘制长期均线（MA50、MA200）、日收益率曲线及直方图、年收益率柱状图、月度收益率热力图，新增数据类型校验与容错处理，解决了原脚本中可能出现的只读数组报错问题，更适合长期行情复盘与收益率统计分析。

- **btc-long-term.py**：专注于比特币（BTC）长期行情的可视化演示，加载比特币历史价格数据，绘制长期K线图、趋势均线等核心图表，适配加密货币长期价格波动大的特点，演示如何通过finplot高效呈现海量历史数据（数十万个数据点），同时简化了分析逻辑，聚焦长期趋势展示，代码简洁易懂，适合新手学习加密货币数据可视化。

- **snp500.py**：标普500指数相关数据的可视化脚本，可加载标普500指数历史价格数据，绘制指数走势、均线、成交量等图表，演示如何处理大盘指数类金融数据，支持趋势分析与简单的波动观察，部分版本还可展示标普500成分股相关联动分析，适配股票大盘数据的可视化场景。

## 二、实时数据类脚本（对接真实交易平台API）

- **bitmex-ws.py**：通过WebSocket协议对接BitMEX加密货币交易平台的实时数据接口，实时获取比特币（XBTUSD等合约）的分时、K线、成交量等实时行情数据，动态更新图表，演示finplot的实时数据渲染能力，同时包含数据格式解析、连接异常处理逻辑，适合学习如何将finplot与实时交易平台API结合，实现加密货币实时行情监控。

- **bfx.py**：对接Bitfinex加密货币交易平台的API（支持REST/WS协议），加载加密货币（如BTC、ETH）的历史及实时行情数据，绘制K线图、成交量图等，演示不同交易平台数据的适配方法，包含数据请求频率控制、异常响应处理，补充了bitmex-ws.py的交易平台覆盖场景，同样聚焦加密货币实时/历史数据可视化。

## 三、动效与交互类脚本（演示finplot动态可视化能力）

- **animate.py**：动态可视化演示脚本，核心功能是生成模拟的K线数据，实现三大动态效果——视图窗口自动平滑移动（模拟行情滚动）、K线低点动态生成标记点、成交量图动态显示数值标签，通过定时器控制动画帧率（默认30FPS），直观展示finplot的实时更新与动画渲染能力，无真实数据依赖，纯演示动态交互效果，也是你之前咨询过的脚本。

- **dockable.py**：交互性增强演示脚本，实现可停靠、可拖拽的子图布局，支持用户手动调整子图位置、大小，新增图表停靠/取消停靠功能，演示finplot的灵活布局能力，适配多维度分析场景下的用户自定义交互需求，让用户可根据自身需求调整图表展示方式。

## 四、高级可视化类脚本（特色图表样式演示）

- **heatmap.py**：热力图可视化演示脚本，核心功能是将金融数据（如月度收益率、价格相关性）转换为热力图，通过颜色深浅直观呈现数据大小或相关性强弱，支持自定义颜色映射、热力图尺寸，演示如何用热力图简化复杂的多维度数据对比，常见应用场景为月度/季度收益率对比、多品种相关性分析。

- **overlay-correlate.py**：叠加对比与相关性分析脚本，支持加载两个及以上金融品种的数据（如两只股票、股票与指数），将其价格走势叠加绘制在同一图表中，计算并展示品种间的相关性，直观对比不同品种的走势差异与联动关系，适配多品种对比分析场景（如个股与大盘、不同加密货币对比）。

- **renko-dark-mode.py**：Renko（砖形图）可视化+暗黑模式演示脚本，核心是绘制Renko图表（一种忽略时间、仅关注价格波动的K线变体，适合识别趋势），同时集成finplot的暗黑主题样式，演示如何自定义图表配色、样式，适配暗黑模式的视觉需求，简化趋势识别难度，聚焦价格波动本身。

- **volume-profile.py**：成交量轮廓（Volume Profile）可视化脚本，绘制价格区间内的成交量分布轮廓，直观展示不同价格区间的成交量密集程度，帮助识别支撑位、压力位（成交量密集区），演示finplot的高级成交量分析功能，适配短线交易中的成交量分布分析场景。

- **bubble-table.py**：气泡图+数据表格联动演示脚本，将金融数据（如股票市值、收益率）以气泡图形式呈现（气泡大小/颜色对应数据维度），同时搭配数据表格展示详细数值，实现图表与表格的联动交互（如点击气泡高亮表格对应数据），演示如何整合多种可视化形式，提升数据解读效率。
